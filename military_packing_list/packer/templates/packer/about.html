Our Mission

The Military Gear Pricer is a community-driven platform designed to help military personnel find the best prices for items on their packing lists. With numerous shops selling gear at varying prices, it can be challenging to get everything you need efficiently and affordably.

How It Works

This application allows users (even without an account) to contribute by adding current prices for specific military gear items from different stores. Here's a quick overview:

  • Browse Items: View a list of common packing list items.
  • View Prices: See prices submitted by other users, sorted by price and submission date to help you find the best deals.
  • Add Prices: Contribute by adding new prices you've found. You can do this anonymously or provide a name.
  • Vote on Submissions: Upvote helpful and accurate price submissions, or downvote outdated or incorrect ones.
  • Reputation: Users who provide names with their submissions earn a reputation score based on the community's votes on their contributions. This helps identify reliable sources of information.

Technology

This application is built using modern web technologies, including:

  • Python (as the primary programming language)
  • Django (as the web framework)
  • PostgreSQL (for the database)
  • Docker & Docker Compose (for containerization and development environment)
  • HTML, CSS, and JavaScript (for the frontend presentation and interactivity)

We aim to provide a simple, effective tool for the military community.
